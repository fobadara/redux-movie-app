### Antiquarian

## Description 🏗️

Antiquarian is a website that keeps track of your books both old and new.

   1. Display a list of books.
   2. Add a book.
   3. Remove a selected book.

## Getting Started 🏁

To get a local copy up and running follow these simple example steps.

> Clone the app by running this command

```bash 
git clone https://github.com/fobadara/antiquarian.git
```
> Make sure you install the dependencies by runing this command

```bash
npm install
```


> View it in your browser by running this command

```bash
 npm start
```
It runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000).

**Prerequisites 📜**

You will be needing:

• A web browser to view output

• A text editor or terminal

**Run tests**

You can run tests by following these steps:

• commit and push your changes

• navigate to pull request

• We have created a pull request which runs linters or you could simply create a pull request of your own
